# Souk AI Agent

## Vision
AI commercial agent that replicates Moroccan souk negotiation experience online.

## Stack
- React + Tailwind (Frontend)
- n8n (Automation Engine)
- Google Sheets (Database)
- Gemini API (AI)
- Gmail SMTP (Notifications)

## Main Features
- Multilingual AI chat (FR / EN / AR)
- Intelligent price negotiation
- Stock verification
- Order processing
- Email notifications
- Admin dashboard
